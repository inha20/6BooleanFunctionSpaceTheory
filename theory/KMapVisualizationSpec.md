## 0. 표준화가 필요한 이유

인지 실험에서 자극 간 **시각적 일관성**은 결과의 해석 가능성을 보장하는 기본 조건이다.
K-map은 세포(cell) 배색, 레이블 위치, 그리드 두께 등 표면적 특성만으로도 "대칭성 감지"에 영향을 줄 수 있다.

이 명세는 아래 두 가지 목표를 동시에 달성하도록 설계됐다:

1. **통제 변인 고정:** 자극 간 색상·레이아웃·폰트 완전 동일
2. **실험 조건 분기:** 버전 V1–V4를 명시적으로 정의해, 실험 조건에 따라 어떤 버전을 사용할지 사전에 지정

## 1. 표준 레이아웃

### 1.1 그리드 구조 (4변수 K-map)

```
       CD=00  CD=01  CD=11  CD=10
AB=00 [  ]   [  ]   [  ]   [  ]
AB=01 [  ]   [  ]   [  ]   [  ]
AB=11 [  ]   [  ]   [  ]   [  ]
AB=10 [  ]   [  ]   [  ]   [  ]
```

- **행(Row):** 변수 AB, Gray code 순서 → 00 · 01 · 11 · 10
- **열(Column):** 변수 CD, Gray code 순서 → 00 · 01 · 11 · 10
- **이유:** 표준 Karnaugh map 배열과 일치시켜 실험 경험이 있는 참가자의 사전 친숙도를 통제한다.

### 1.2 최소항 번호 (Minterm Index)

Gray code 순서에 따른 16개 셀의 최소항 번호 배치:

```
       CD=00  CD=01  CD=11  CD=10
AB=00   m0     m1     m3     m2
AB=01   m4     m5     m7     m6
AB=11  m12    m13    m15    m14
AB=10   m8     m9    m11    m10
```

최소항 번호는 버전 V1에서만 표시된다 (§3 참조).

| 요소 | 색상 코드 | 설명 |
|------|-----------|------|
| **1-셀 (채움)** | `#2D2D2D` | 진한 회색 (black 대비 눈부심 감소) |
| **0-셀 (빈칸)** | `#FFFFFF` | 흰색 |
| **그리드 선** | `#000000` | 검정 |
| **헤더 배경** | `#F0F0F0` | 연한 회색 |
| **헤더 텍스트** | `#000000` | 검정 |

| **셀 값 (0/1)** | `#000000` / `#FFFFFF` | 1-셀 내부 흰색, 0-셀 내부 검정 |

> **색상 선택 근거:**
> - 1-셀 `#2D2D2D`: 순수 검정(`#000000`)은 인쇄 시 번짐이 있고 화면에서 과도하게 강함. 진한 회색은 동등한 대비를 유지하면서 시각적 피로를 줄인다.
> - 0-셀 흰색: 최대 대비를 통해 패턴 구조의 명확한 인식을 유지한다.

## 5. 자극 버전 정의 (V1–V4)

실험 조건에 따라 네 가지 버전을 사전 정의한다.

| 버전 | 헤더 레이블 | 셀 값 (0/1) | 최소항 번호 | 사용 실험 |
|------|-----------|------------|------------|---------|
| **V1 — 완전 표시** | ✅ | ✅ | ✅ | 사전 친숙도 측정, 대조 조건 |
| **V2 — 레이블+패턴** | ✅ | ✅ | ❌ | Exp-2 (연구 가치 판단) |
| **V3 — 레이블+색상** | ✅ | ❌ | ❌ | Exp-1 (주의 순서 측정) 기본 |
| **V4 — 순수 패턴** | ❌ | ❌ | ❌ | Exp-1 배경 지식 통제 조건 |

### 각 버전 권고 이유

- **V3를 Exp-1 기본으로:** "패턴"을 보는 경험을 최대화하고 숫자(0/1) 읽기가 주의를 방해하지 않도록 한다. 헤더는 유지해 변수 구조 인식을 허용한다.
- **V2를 Exp-2에:** 연구 가치 판단(H4)은 패턴의 의미를 이해한 후에 이루어지므로, 셀 값 표시로 정보를 제공한다. 최소항 번호는 과부하를 줄이기 위해 생략.
- **V4를 통제 조건으로:** 헤더 없이 순수 흑백 패턴만 제시 → 수학적 배경이 없는 참가자도 "시각적 패턴"만으로 반응할 수 있음.

## 6. 기준 자극 명세 (Reference Stimuli)

`CognitiveExperimentDesign.md §1.2`의 S₄-불동점 10개 핵심 자극을 예시로 완전 기술한다.

### 6.1 F01 — f = 0 (상수 0)

```
모든 16개 셀 = 0 (흰색)
```

| | CD=00 | CD=01 | CD=11 | CD=10 |
|---|---|---|---|---|
| AB=00 | 0 | 0 | 0 | 0 |
| AB=01 | 0 | 0 | 0 | 0 |
| AB=11 | 0 | 0 | 0 | 0 |
| AB=10 | 0 | 0 | 0 | 0 |

- S₄-불동점: ✅ (자명)
- 궤도 크기: 1 (모든 치환에 불변)
- 시각적 특징: 완전 공백 — 대칭의 극단

### 6.2 F02 — f = 1 (상수 1)

```
모든 16개 셀 = 1 (진한 회색)
```

| | CD=00 | CD=01 | CD=11 | CD=10 |
|---|---|---|---|---|
| AB=00 | 1 | 1 | 1 | 1 |
| AB=01 | 1 | 1 | 1 | 1 |
| AB=11 | 1 | 1 | 1 | 1 |
| AB=10 | 1 | 1 | 1 | 1 |

- S₄-불동점: ✅
- 궤도 크기: 1
- 시각적 특징: 완전 채움 — 대칭의 극단

### 6.3 F03 — A⊕B⊕C⊕D (체커보드)

```
교번(alternating) 패턴 — 인접 셀이 항상 다름
```

| | CD=00 | CD=01 | CD=11 | CD=10 |
|---|---|---|---|---|
| AB=00 | 0 | 1 | 0 | 1 |
| AB=01 | 1 | 0 | 1 | 0 |
| AB=11 | 0 | 1 | 0 | 1 |
| AB=10 | 1 | 0 | 1 | 0 |

- S₄-불동점: ✅ (Space(XOR)의 핵심 자극)
- 궤도 크기: 1
- 시각적 특징: 체커보드 — 이 연구 프로그램의 최초 발견 자극

### 6.4 F04 — ¬(A⊕B⊕C⊕D) (역 체커보드)

F03의 보수(complement). 1-0 반전.

| | CD=00 | CD=01 | CD=11 | CD=10 |
|---|---|---|---|---|
| AB=00 | 1 | 0 | 1 | 0 |
| AB=01 | 0 | 1 | 0 | 1 |
| AB=11 | 1 | 0 | 1 | 0 |
| AB=10 | 0 | 1 | 0 | 1 |

- S₄-불동점: ✅
- 궤도 크기: 1
- 시각적 특징: F03의 거울상 (색 반전)

### 6.5 F05 — Threshold-1: HW ≥ 1 (= A∨B∨C∨D)

0은 minterm m0 하나뿐.

| | CD=00 | CD=01 | CD=11 | CD=10 |
|---|---|---|---|---|
| AB=00 | 0 | 1 | 1 | 1 |
| AB=01 | 1 | 1 | 1 | 1 |
| AB=11 | 1 | 1 | 1 | 1 |
| AB=10 | 1 | 1 | 1 | 1 |

- S₄-불동점: ✅
- 시각적 특징: 좌상단 한 칸만 흰색

### 6.6 F06 — Threshold-2: HW ≥ 2

HW = 0 또는 1인 셀은 0 (m0, m1, m2, m4, m8).

| | CD=00 | CD=01 | CD=11 | CD=10 |
|---|---|---|---|---|
| AB=00 | 0 | 0 | 1 | 0 |
| AB=01 | 0 | 0 | 1 | 1 |
| AB=11 | 1 | 1 | 1 | 1 |
| AB=10 | 0 | 1 | 1 | 1 |

> ⚠️ Gray code 배열로 인해 HW에 따른 시각 패턴이 직관적이지 않을 수 있음. 이 자극은 V1(최소항+값 표시) 버전 사용 권장.

- S₄-불동점: ✅ (HW ≥ 2: 대칭 임계값 함수)
- 시각적 특징: 대각선 블록 구조

### 6.7 F07 — Threshold-3: HW ≥ 3

HW ≥ 3인 셀 (m7, m11, m13, m14, m15)만 1.

| | CD=00 | CD=01 | CD=11 | CD=10 |
|---|---|---|---|---|
| AB=00 | 0 | 0 | 0 | 0 |
| AB=01 | 0 | 0 | 1 | 0 |
| AB=11 | 0 | 1 | 1 | 1 |
| AB=10 | 0 | 0 | 1 | 1 |

- S₄-불동점: ✅
- 시각적 특징: 우하단 집중

### 6.8 F08 — Threshold-4 = A∧B∧C∧D

오직 m15만 1.

| | CD=00 | CD=01 | CD=11 | CD=10 |
|---|---|---|---|---|
| AB=00 | 0 | 0 | 0 | 0 |
| AB=01 | 0 | 0 | 0 | 0 |
| AB=11 | 0 | 0 | 1 | 0 |
| AB=10 | 0 | 0 | 0 | 0 |

- S₄-불동점: ✅
- 시각적 특징: 우하단 한 칸만 채워짐 (F05의 쌍대)

## 7. 통제 자극 (Control Stimuli)

### 7.1 비구조적 통제 (Random Control)

실험 자극과 시각적으로 구분되지 않도록 **밀도(density)를 매칭**한다.

| 자극 유형 | 권장 생성 방법 |
|---------|-------------|
| 저밀도 랜덤 (1–4셀) | 랜덤 4셀 선택 — F08 밀도와 매칭 |
| 중밀도 랜덤 (6–10셀) | 랜덤 8셀 선택 — 중간 임계값 함수와 밀도 매칭 |
| 고밀도 랜덤 (12–15셀) | 랜덤 14셀 선택 — F05 밀도와 매칭 |

> 랜덤 자극은 실험 전 Python 스크립트로 생성하고 시드(seed)를 기록해 재현 가능성을 보장할 것.

### 7.2 "구조 있지만 불동점 아님" 통제

S₄-불동점이 아닌 구조적 자극 — Exp-3의 핵심 대조 조건.

| 예시 함수 | 궤도 크기 | 구조적 이유 |
|---------|---------|---------|
| A∧B | 6 | 두 변수 AND — S₄하에 6개 등가 |
| A∧B∧C | 4 | 세 변수 AND |
| (A∧B)∨(C∧D) | 3 | 두 쌍 AND-OR |

## 9. 제작 방법 지침

> **이미지 생성 규칙 (HARD RULE, SESSION_START.md §3에서 이관):**
> **허용:** PowerPoint, Keynote, Draw.io, 연구자가 직접 작성한 SVG.  
> **절대 금지:** AI 이미지 생성 도구(DALL-E, SD 등) 사용 금지. AI가 작성한 Python/matplotlib/LaTeX/TikZ 코드로 그림 생성 금지.

### 9.1 PowerPoint 제작 가이드라인

1. 슬라이드 크기: 10cm × 10cm (정사각형)
2. 표(Table) 삽입: 5행 × 5열 (헤더 1행 + 데이터 4행; 헤더 1열 + 데이터 4열)
3. 셀 채우기: 수동으로 `#2D2D2D` (1-셀) 또는 흰색 (0-셀)
4. 텍스트: 명세 §4의 폰트 및 크기 적용
5. 내보내기: PNG, 300 DPI

### 9.2 Draw.io 제작 가이드라인

1. `File → Page Setup` → 400px × 400px
2. 격자 선: `Format → Line Style` → 2px, 검정
3. 셀: `Fill Color`로 색상 직접 입력 (`#2D2D2D` / `#FFFFFF`)
4. 텍스트: 폰트 직접 지정 (Default font 대신 Arial 지정)
5. `File → Export as → PNG` → Scale 1x
