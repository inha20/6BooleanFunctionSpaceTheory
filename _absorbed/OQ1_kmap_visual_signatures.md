## 1. 질문 진술

> **OQ-1:** 다양한 Space(G)에 속하는 함수들이 카르노 맵(Karnaugh Map) 위에서  
> 각 공간만의 특유한 **시각적 시그니처(Visual Signature)**를 가지는가?  
> 만약 그렇다면, 그 시그니처를 체계적으로 기술하고 분류하라.

이 질문은 `FunctionSpaceTheory.md`, `KarnaughGeometry.md`에서 제기된 핵심 미해결 문제이며,  
연구 프로그램 전체(SRT, Repo 1, Repo 3)와 직결되는 교차점이다.

## 2. 동기 및 배경

### 2.1 왜 K-map 시각적 시그니처인가?

4변수 Boolean 함수 65,536개는 4×4 격자의 0/1 패턴으로 완전히 기술된다.  
Space(G)의 경우 G가 달라지면 속하는 함수들의 집합이 달라지며,  
**그 집합이 K-map 위에서 공통된 기하학적 특성을 가진다**면:

1. 새로운 함수 f를 보고 "이 함수가 어느 Space(G)에 속하는가?"를 **시각적으로 판단** 가능
2. K-map의 패턴 자체가 **대수적 클론 구조의 직관적 표현**이 됨
3. SRT H3(주의 필터)와 직결: 연구자의 시각적 주의를 끄는 구조가 실제로 대수적 불변성과 일치

### 2.2 기존 관찰

이미 알려진 시각적 특성:
- **Space(XOR)**: 체커보드 또는 줄무늬 패턴 (XOR_affine_space.md)
- **Space(AND)**: 단조 패턴 — 1-영역이 "더 많은 1 입력" 쪽에 집중
- **Space(NAND)**: 모든 패턴 포함 (시그니처 없음 = 무제약)

## 3. 각 Space(G)의 시각적 시그니처 분석

### 3.1 Space(XOR) — 아핀 함수 (32개)

**K-map 특성:**
```
특성 1: 체커보드 또는 선형 줄무늬
특성 2: 1-영역의 크기 = 항상 2의 거듭제곱 (0, 2, 4, 8, 16개)
특성 3: ANF에서 AND 항이 없음 → K-map의 어떤 2×2 부분격자에서도 짝수 개의 1
```

**대표 패턴:**
```
f = A⊕B (줄무늬):
        CD
        00  01  11  10
AB  00 | 0 | 0 | 0 | 0 |
    01 | 1 | 1 | 1 | 1 |
    11 | 0 | 0 | 0 | 0 |
    10 | 1 | 1 | 1 | 1 |

f = A⊕B⊕C⊕D (완전 체커보드):
        CD
        00  01  11  10
AB  00 | 0 | 1 | 0 | 1 |
    01 | 1 | 0 | 1 | 0 |
    11 | 0 | 1 | 0 | 1 |
    10 | 1 | 0 | 1 | 0 |
```

> **시그니처 정의 (아핀):** 4×4 K-map을 임의의 2×2 부분격자(환면체 위 인접 4칸)로 볼 때,  
> 각 부분격자에서 1의 개수가 항상 **짝수(0, 2, 4)**이다.

*명세: 아핀 함수 6가지 K-map, 각 2×2 부분격자에 1의 수 표시*

### 3.2 Space(AND) / Space(OR) — 단조 함수 부분집합

**단조 함수 정의:** x ≤ y (성분별) → f(x) ≤ f(y)  
(어떤 입력 비트를 0→1로 바꿔도 출력이 1→0으로 바뀌지 않음)

**K-map 특성:**
```
특성 1: 1-영역이 "위쪽"(더 많은 변수가 1인 쪽) — 경계선이 단조
특성 2: K-map에서 임의의 인접 쌍 (칸 A, 칸 B)에서 A→B로 이동 시
         A의 1수가 B보다 작으면 f(A) ≤ f(B)
특성 3: 0→1 전환만 허용, 1→0 전환 금지 (단조성)
```

**예시 (f = AB — AND):**
```
        CD
        00  01  11  10
AB  00 | 0 | 0 | 0 | 0 |
    01 | 0 | 0 | 0 | 0 |
    11 | 1 | 1 | 1 | 1 |
    10 | 0 | 0 | 0 | 0 |
```

> 참고: 4변수 단조 함수 전체 수는 Dedekind 수 D(4) = 168개.  
> Space(AND)와 Space(OR)는 각각 168개 중 AND/OR로 생성 가능한 부분집합.

[그림 플레이스홀더: OQ1_Figure02_monotone_signatures.png]  
*명세: 단조 함수 6가지 K-map, 단조 경계선 강조*

### 3.3 Space(NOT + id) — 자기쌍대 함수

**자기쌍대 정의:** f(x₁,...,xₙ) = ¬f(¬x₁,...,¬xₙ)

**K-map 특성:**
```
특성 1: K-map에서 "대각 반전 대칭": 위치 i와 그 보수 위치 ¬i에서 값이 서로 반대
특성 2: 항상 정확히 8개의 1 (4변수 기준)
특성 3: 180° 회전(환면체 위에서) 시 0↔1 교환
```

**예시 (f = Majority(A,B,C,D) 변형):**
```
4변수 자기쌍대 함수는 16개 입력 쌍 중 8쌍을 1로 → 나머지 8개는 자동으로 0
```

> **시그니처 정의 (자기쌍대):** K-map의 각 (위치, 보수 위치) 쌍에서 값이 정확히 하나만 1.

[그림 플레이스홀더: OQ1_Figure03_selfdual_signatures.png]  
*명세: 자기쌍대 함수 4가지, 보수 위치 쌍 화살표 표시*

### 3.4 Space(NAND) = Space(NOR) = Ω

- 모든 65,536개 패턴이 포함됨
- **시각적 시그니처 없음** — 어떤 패턴도 이 공간에 속함
- 시그니처의 "부재"가 곧 NAND의 만능성을 나타냄

## 4. 시그니처 판별 알고리즘 (OQ-1 실용적 응용)

임의의 K-map f가 주어졌을 때:

```
Step 1: 아핀 판별
  → 모든 2×2 부분격자에서 1의 수가 짝수인가?
  → Yes: f ∈ Space(XOR) (아핀)
  → No: 다음 단계

Step 2: 단조 판별
  → K-map에서 인접 비교 시 1→0 전환이 없는가?
  → Yes: f ∈ Space(AND) 또는 Space(OR) 후보 (단조)
  → No: 다음 단계

Step 3: 자기쌍대 판별
  → 각 (위치, 보수 위치) 쌍에서 값이 반대인가?
  → Yes: f는 자기쌍대 (Space(NOT+id) 후보)
  → No: 다음 단계

Step 4: 모든 판별 통과 못하면
  → f ∈ Space(NAND) = Ω, 다른 특수 Space(G)에는 속하지 않음
```
