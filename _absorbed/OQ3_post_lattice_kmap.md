## 1. 질문 진술

> **OQ-3:** Post의 격자(Post's Lattice)에 등장하는 각 클론(닫힌 함수 클래스)은  
> 카르노 맵 기하학(Karnaugh Map Geometry)을 통해 어떻게 시각화할 수 있는가?  
> 격자 위의 포함 관계가 K-map 패턴의 포함 관계와 어떻게 대응되는가?

이 질문은 `PostLattice.md`에서 제기된 핵심 미해결 문제이며, Post 이론의 추상적 내용을  
K-map이라는 **직관적이고 시각적인 언어**로 번역하는 시도이다.

## 2. Post 격자 복습

### 2.1 5가지 핵심 Post 클래스

| 기호 | 이름 | 정의 | 판별 조건 |
|-----|-----|-----|--------|
| **T₀** | 0-보존 | f(0,...,0) = 0 | K-map의 (0,0,0,0) 위치 = 0 |
| **T₁** | 1-보존 | f(1,...,1) = 1 | K-map의 (1,1,1,1) 위치 = 1 |
| **S** | 자기쌍대 | f(¬x) = ¬f(x) | 각 (위치, 보수) 쌍 값 반대 |
| **M** | 단조 | x ≤ y → f(x) ≤ f(y) | K-map에서 1→0 전환 없음 |
| **L** | 선형(아핀) | ANF 차수 ≤ 1 | 모든 2×2 부분격자에서 1의 수 짝수 |

### 2.2 Post 격자의 핵심 구조 (간략화)

```
Ω (전체 65,536개)
│
├─ T₀ (0 보존)        ├─ T₁ (1 보존)
│   └─ M∩T₀ (단조+0보존)   └─ M∩T₁ (단조+1보존)
│       └─ M∩T₀∩T₁ (양쪽 보존)
│
├─ M (단조)
│
├─ L (선형/아핀)
│   └─ L∩T₀        └─ L∩T₀∩T₁
│       └─ L∩T₀∩T₁∩M
│
├─ S (자기쌍대)
│   └─ S∩L
│
└─ ... (무한히 많은 클론이 있으나, 위가 주요 유한 클론)
```

## 3. 각 Post 클래스의 K-map 기하학

### 3.1 T₀ — 0-보존 함수

**K-map 특성:**
- 진리표 위치 0000 (K-map 좌상단 00/00) = 반드시 **0**
- 반대로 말하면: K-map에서 "모든 입력이 0인 코너"가 0인 함수들

```
T₀ K-map 제약:
      CD
      00  01  11  10
AB 00| 0 | ? | ? | ? |   ← 이 칸 = 0 (고정)
   01| ? | ? | ? | ? |
   11| ? | ? | ? | ? |
   10| ? | ? | ? | ? |
```

4변수 T₀ 함수의 수: 2^{16-1} = 2^{15} = 32,768개 (절반)

### 3.2 T₁ — 1-보존 함수

**K-map 특성:**
- K-map의 우하단 영역(AB=11, CD=11) 위치 = 반드시 **1**

```
T₁ K-map 제약:
      CD
      00  01  11  10
AB 00| ? | ? | ? | ? |
   01| ? | ? | ? | ? |
   11| ? | ? | 1 | ? |   ← 이 칸(1111) = 1 (고정)
   10| ? | ? | ? | ? |
```

4변수 T₁ 함수의 수: 2^{15} = 32,768개

### 3.3 M — 단조 함수

**K-map 특성:**  
K-map에서 "인접 칸으로 이동 시 1→0 전환이 없음"

```
단조 함수 제약:
  격자 위에서 아래로 이동하거나 오른쪽으로 이동할 때
  (= 변수 하나가 0→1로 변할 때)
  출력값은 0→1만 허용, 1→0 불허
```

**시각적 특징:**  
1-영역이 K-map의 "높은 쪽"(입력 1이 많은 쪽)에만 존재  
0-영역과 1-영역의 경계가 단조적

[그림 플레이스홀더: OQ3_Figure01_monotone_boundary.png]  
*명세: 단조 함수의 K-map 3가지, 1-영역 경계에 화살표 방향(→ 단조 증가) 표시*

### 3.4 L — 선형/아핀 함수

**K-map 특성:**  
임의의 2×2 부분격자(환면체 위에서 인접한 4칸)의 XOR 합 = 0

```
아핀 조건 (2×2 부분격자 검사):
모든 (a,b), (a,b+1), (a+1,b), (a+1,b+1) 위치에 대해:
  f(a,b) ⊕ f(a,b+1) ⊕ f(a+1,b) ⊕ f(a+1,b+1) = 0
```

이는 ANF에서 2차 이상 항이 없다는 조건과 동치이다.

**시각적 특징:** 체커보드 또는 균일한 줄무늬 패턴

### 3.5 S — 자기쌍대 함수

**K-map 특성:**  
K-map에서 위치 i와 그 "보수 위치" (15-i)에서 값이 정확히 반대

```
4변수 보수 관계:
위치 0(0000) ↔ 위치 15(1111)  → f(0000) ≠ f(1111)
위치 1(0001) ↔ 위치 14(1110)  → f(0001) ≠ f(1110)
위치 2(0010) ↔ 위치 13(1101)  → f(0010) ≠ f(1101)
...
```

**시각적 특징:** 180° 회전 대칭 (환면체 위에서) + 값 교환

[그림 플레이스홀더: OQ3_Figure02_selfdual_rotation.png]  
*명세: 자기쌍대 함수의 K-map, 보수 쌍 위치에 이중 화살표로 표시*

## 4. 포함 관계의 K-map 기하학적 해석

### 4.1 클래스 교집합의 K-map 제약 누적

Post 격자에서 아래로 내려갈수록(더 제약이 많아질수록) K-map 제약이 누적된다:

| 클래스 | K-map 제약 | 4변수 함수 수 |
|-------|---------|-----------|
| Ω | 없음 | 65,536 |
| T₀ | 코너(0,0,0,0) = 0 | 32,768 |
| T₀ ∩ T₁ | 두 코너 고정 | 16,384 |
| M | 단조 경계 | 168 (Dedekind D(4)) |
| M ∩ T₀ ∩ T₁ | 단조 + 두 코너 고정 | ≤ 168 |
| L | 2×2 짝수 XOR 조건 | 32 |
| L ∩ T₀ | 아핀 + (0,...,0)=0 | 16 |
| L ∩ T₀ ∩ T₁ | 아핀 + 두 코너 고정 | 8 |
| S | 180° 반점 대칭 | 256 (= 2^8) |
| S ∩ L | 아핀 + 자기쌍대 | 2 (f=A⊕B⊕C⊕D만?) |

> ⚠️ S ∩ L의 정확한 수는 확인 필요 (FQ-3 관련)

### 4.2 K-map 제약의 위계

```
제약 없음 (Ω)
    ↓
코너 값 고정 (T₀, T₁)
    ↓
전역 단조성 조건 (M)  또는  전역 아핀 조건 (L)
    ↓
단조 + 아핀 (M ∩ L = 선형 단조)  또는  아핀 + 자기쌍대 (S ∩ L)
    ↓
상수 함수 {0, 1}
```
