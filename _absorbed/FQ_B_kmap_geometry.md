## FQ-6. Geometric Patterns of Linear/Monotone/Self-Dual Functions on Karnaugh Maps

### 질문
선형(아핀), 단조, 자기쌍대 함수의 카르노 맵 위 기하학적 패턴을 체계화하라.

### 분석

#### 6.1 선형(아핀) 함수의 K-map 기하학

**정의:** ANF에서 AND 항 없음 → $f = a_0 \oplus a_1 A \oplus a_2 B \oplus a_3 C \oplus a_4 D$

**기하학적 특성:**
- **"균형(Balance)":** 변수 개수가 1개 이상이면 1의 개수 = 0의 개수 = 8개 (4변수)
  - 예외: 상수 0 (0개), 상수 1 (16개)
- **"아핀 균일성":** 4×4 K-map을 2×2 부분격자로 분할 시 각 부분격자에서 1의 수가 항상 짝수
- **"선형 경계":** 1-영역과 0-영역의 경계가 직선(K-map의 행/열/대각선)

**패턴 분류 (4변수 32개):**

```
유형 1: 상수 (2개)
  f=0: 모두 0  /  f=1: 모두 1

유형 2: 단일 변수 투영 (8개: A, B, C, D와 각각의 보수)
  수평 줄무늬(A 또는 C 기준) 또는 수직 줄무늬(B 또는 D 기준)

유형 3: 두 변수 XOR (12개: C(4,2)×2)
  교대 블록 패턴 — 2×1, 1×2, 혹은 대각 교대

유형 4: 세 변수 XOR (8개: C(4,3)×2)
  체커보드에 가까운 패턴

유형 5: 네 변수 XOR (2개: A⊕B⊕C⊕D, ¬(A⊕B⊕C⊕D))
  완전 체커보드
```

*명세: 5가지 유형 대표 K-map 2×3 갤러리, 유형 레이블 표시*

#### 6.2 단조 함수의 K-map 기하학

**기하학적 특성:**
- **"위쪽 닫힘(Upward Closure)":** K-map에서 1-영역은 항상 "상위 원소를 포함"
  - 즉, 1이 표시된 칸의 "Hamming 상위 원소"도 반드시 1
- **"경계의 단조성":** 0→1 전환 경계는 변수 증가 방향으로만 이동

**단조 D(4)=168개 분류:**
- 168개 = 전체 65,536개 중 0.26%
- 임의의 두 단조 함수의 AND, OR는 단조 → 단조 클래스는 AND/OR에 닫힘

#### 6.3 자기쌍대 함수의 K-map 기하학

**기하학적 특성:**
- **"반점 대칭(Antipodal Symmetry)":** K-map의 각 위치 i와 그 보수 위치 15-i에서 값이 반대
- **"고정 1의 수 = 8":** 4변수 자기쌍대 함수는 정확히 8개의 1을 가짐
  - 4변수 자기쌍대 함수의 수: $2^{2^{n-1}} = 2^8 = 256$개

## FQ-7. Visual Invariant for Space(G) — Do All Functions Share a Common Pattern?

### 질문
Space(G) 내의 모든 함수가 카르노 맵 위에서 공통 시각적 불변량(visual invariant)을 공유하는가?

### 분석

#### 7.1 "시각적 불변량"의 정의

**정의 (Visual Invariant):**  
집합 S의 시각적 불변량 P란, 모든 f ∈ S에 대해 f의 K-map이 P를 만족하고,  
어떤 g ∉ S에 대해서는 g의 K-map이 P를 만족하지 않는 조건.

즉, P는 S의 K-map 특성화(characterization)이다.

#### 7.2 각 Space(G)에 대한 시각적 불변량

| Space(G) | 시각적 불변량 P | 완전 특성화? |
|---------|-------------|-----------|
| Ω | 없음 (모든 패턴) | ✅ (자명) |
| L (아핀) | 모든 2×2 부분격자에서 1의 수 짝수 | ✅ 완전 특성화 |
| M∩T₀∩T₁ (단조) | 인접 칸 1→0 전환 없음 | ✅ 완전 특성화 |
| S (자기쌍대) | 반점 대칭 | ✅ 완전 특성화 |
| L ∩ M | 선형 + 단조 조건 동시 | ✅ |

**핵심 결론:**  
모든 주요 Post 클래스에 대해 시각적 불변량이 존재한다.  
→ **K-map = Post 클래스 분류의 완전한 시각적 인터페이스**

#### 7.3 불변량의 검증 가능성

각 불변량은 K-map을 보는 것만으로 O(n²) 시간에 확인 가능:
- 아핀: 모든 2×2 부분격자 검사 → O(16) = O(1) for n=4
- 단조: 모든 인접 쌍 검사 → O(n·2ⁿ)
- 자기쌍대: 모든 보수 쌍 검사 → O(2^{n-1})

## FQ-8. Is the Repo 1 Checkerboard Pattern the Geometric Expression of Space(XOR)?

### 질문
Repo 1 (KMapStructureInvariance)에서 발견된 체커보드 패턴이 Space(XOR)의 기하학적 표현인가?

### 분석 — 가설 검증

#### 8.1 주장

**주장 (FQ-8):** Repo 1의 체커보드 불변성 결과 = Space(XOR) = L (아핀 클론)의  
K-map 기하학적 표현이다.

#### 8.2 근거 1: 체커보드 = XOR 함수의 K-map

$$A \oplus B \oplus C \oplus D = 1 \iff \text{K-map에서 체커보드 패턴}$$

- 이 함수는 Space(XOR)의 "대표 원소(representative element)" 중 하나
- S₄ 작용 하에 이 함수는 자기 자신으로 매핑됨 (완전 대칭)

#### 8.3 근거 2: 아핀 공간 = 체커보드의 선형 변환 궤도

Space(XOR)의 32개 함수는 체커보드에서 다음 변환들로 얻어진다:
- 변수 제거: A⊕B⊕C⊕D에서 xᵢ 항 제거 → 줄무늬 패턴
- 상수 XOR: f ⊕ 1 → 0과 1 위치 교환

이는 K-map 위에서:
- 체커보드 → 줄무늬로의 "투영" = 변수 제거
- 체커보드 → 역 체커보드 = ⊕1

#### 8.4 근거 3: 변수 치환 불변성

Repo 1의 핵심 결과: "특정 함수들이 변수 치환(σ ∈ S₄) 하에 불변"
Space(XOR)의 핵심 성질: S₄ 작용 하에 닫혀 있음 (클론의 기본 속성)

**가설 확인:**  
Repo 1에서 발견된 "변수 치환 불변 함수들의 집합" ⊆ Space(XOR)?  
→ 체커보드 패턴 자체는 S₄ 작용의 불동점(fixed point) → ✅ Space(XOR) ∈ 자명

#### 8.5 결론 및 한계

**확인된 부분:** 체커보드(A⊕B⊕C⊕D)는 Space(XOR)의 원소이며, K-map 기하학의 시각적 대표원소

**미확인 부분:** Repo 1의 "불변 구조 결과"가 정확히 어떤 집합을 가리키는지 확인 필요  
→ 해당 집합이 정확히 Space(XOR) = L과 일치하는지는 Repo 1 원문 재검토 필요

## FQ-9. Variable Rearrangement (Repo 3) vs. Gate-Generated Space Classification

### 질문
Repo 3의 변수 재배열 불변성 분류와 게이트 생성 공간 분류는 어떤 관계인가?

### 분석

#### 9.1 두 분류의 비교

**Repo 3의 분류:**
- 기준: σ ∈ S₄ (변수 순열) 하에서 f ~ g ⟺ g = f∘σ
- 결과: 65,536개 함수가 S₄ 궤도로 분류

**Repo 6의 분류:**
- 기준: f가 어느 Space(G)에 속하는가?
- 결과: 함수가 Gate 클론별로 분류

#### 9.2 관계의 구조

```
함수 f ∈ Space(G)
          ↓ S₄ 작용
함수 f∘σ ∈ Space(G) (∵ Space(G)는 S₄-불변)
```

→ **Space(G) 내에서 S₄ 궤도를 분석하는 것이 의미 있음**

**Space(XOR)의 S₄ 궤도 (OQ-2에서 분석):**
10개 궤도 → 각 궤도가 "몇 개 변수가 포함된 XOR 패턴인가"로 해석됨

#### 9.3 두 분류의 정제(Refinement) 관계

Repo 6 분류가 Repo 3 분류보다 "더 거친(coarser)" 분류이다:
- Repo 6: f와 g가 같은 Space(G)에 속하면 같은 "게이트 타입"
- Repo 3: f와 g가 같은 S₄ 궤도에 속하면 "대칭 동치"
- 한 S₄ 궤도 ⊂ 한 Space(G) (Space(G)가 S₄ 불변이므로)
- 역방향: 한 Space(G) = 여러 S₄ 궤도의 합집합

**정제 다이어그램:**
```
Space(XOR) = 궤도₁ ∪ 궤도₂ ∪ ... ∪ 궤도₁₀
Space(AND) = 궤도_M₁ ∪ 궤도_M₂ ∪ ...
```

→ Repo 3 분류 = Repo 6 분류의 세밀한 버전  
→ **논문 Paper 6의 핵심 주장으로 발전 가능**

#### 9.4 미해결 연결

Repo 3의 분류 결과 중:
- "대칭성이 높은 함수"(궤도 크기 < 24인 함수) = 어느 Space(G)에 주로 속하는가?
- Space(XOR)의 10개 궤도 중 Repo 3에서 "불변 구조"로 등장하는 것은?
