## 2. 배경: 두 가지 분류 체계

### 2.1 축 교환 대칭에 의한 분류 (Repo 3)

4변수 Boolean 함수에서 K-map의 **변수 순서 재배열**을 고려한 분류:

**군(Group):** 변수 치환군 S₄ (4개 변수 A, B, C, D의 모든 순열)  
**동치 관계:** f ~ g ⟺ ∃σ ∈ S₄ such that g(x) = f(σ(x))  
**동치류:** S₄ 작용 하의 궤도(Orbit)

특히 "축 교환(Axis-swap)"은 K-map의 수평/수직 축을 교환하는 것:
- AB↔CD 교환: F(A,B,C,D) ↦ F(C,D,A,B)
- 이는 S₄의 특정 원소에 해당

### 2.2 게이트 공간에 의한 분류 (Repo 6)

동일한 4변수 함수를 게이트가 생성하는 공간으로 분류:

```
Space(NAND) = Ω  ⊃  Space(AND)  ⊃  ...
Space(XOR) = 아핀 공간 (32개)
Space(AND) = 단조 공간 부분집합
...
```

## 3. 핵심 명제: Space(G)는 변수 치환에 닫혀 있다

**명제 3.1:** f ∈ Space(G) 이고 σ ∈ S₄ 이면 f∘σ ∈ Space(G).

**증명 스케치:**
- Space(G)의 정의(클론): f를 G-게이트 회로로 구현할 수 있음
- 변수 치환 σ를 적용하면 같은 회로에서 입력 와이어를 재배치하는 것
- 회로 구조가 G 게이트만으로 이루어져 있다는 사실은 변하지 않음
- 따라서 f∘σ도 G-게이트만으로 구현 가능 → f∘σ ∈ Space(G) ∎

**결론:** 모든 Space(G)는 S₄-불변이다.  
→ Space(G)는 S₄ 작용에 의해 궤도(Orbit)들의 합집합으로 분해된다.

## 4. S₄ 궤도와 Space(G)의 관계

### 4.1 아핀 공간(Space(XOR))의 S₄ 궤도 구조

Space(XOR)는 32개 함수로 구성. S₄(크기 24)의 작용 하에:

| 대표 함수 | 궤도 크기 | 설명 |
|---------|---------|-----|
| 0 (상수) | 1 | 변수 없음 |
| 1 (상수) | 1 | 변수 없음 |
| A | 4 | 단일 변수 {A, B, C, D} |
| A⊕1 (=¬A) | 4 | 단일 변수의 보수 |
| A⊕B | 6 | 두 변수 XOR (C(4,2)=6개) |
| A⊕B⊕1 | 6 | |
| A⊕B⊕C | 4 | 세 변수 XOR |
| A⊕B⊕C⊕1 | 4 | |
| A⊕B⊕C⊕D | 1 | 완전 패리티 (체커보드) |
| A⊕B⊕C⊕D⊕1 | 1 | 체커보드의 보수 |

> 궤도 크기의 합: 1+1+4+4+6+6+4+4+1+1 = **32** ✓

**핵심 관찰:** Space(XOR)는 S₄ 작용 하에 **10개의 궤도**로 분해된다.  
각 궤도는 "몇 개의 변수가 포함된 XOR 함수"라는 구조적 의미를 가진다.

### 4.2 축 교환(AB↔CD)과 아핀 공간

A⊕B에 축 교환(A↔C, B↔D)을 적용:  
→ C⊕D — 여전히 Space(XOR) 안에 있음 ✓

A⊕B⊕C에 축 교환 적용:  
→ C⊕D⊕A — 역시 Space(XOR) 안에 있음 ✓

→ **아핀 공간은 축 교환 대칭에 대해 닫혀 있음** (명제 3.1의 특수 사례)

### 4.3 연결 가설

**가설 OQ-2:**  
Repo 3에서 정의된 변수 재배열 동치류와 Repo 6의 Space(G) 분류는 다음 관계를 가진다:

> 두 함수 f, g가 같은 Space(G)에 속하더라도, 반드시 같은 S₄ 동치류에 속하지는 않는다.  
> 그러나 Space(G)는 완전한 S₄ 궤도들의 합집합으로만 표현된다.

이는 "Space(G) 내의 궤도 구조"를 연구하는 것이 의미 있음을 시사한다.

## 5. 축 교환 동치류와 K-map 시각적 대응

### 5.1 아핀 함수에서 축 교환

```
A⊕B의 K-map (수평 줄무늬):        C⊕D의 K-map (수직 줄무늬):
      CD                                   CD
      00 01 11 10                          00 01 11 10
AB 00| 0  0  0  0 |              AB 00| 0  1  0  1 |
   01| 1  1  1  1 |                 01| 0  1  0  1 |
   11| 0  0  0  0 |                 11| 0  1  0  1 |
   10| 1  1  1  1 |                 10| 0  1  0  1 |
```

축 교환(AB↔CD) = K-map 위에서 행-열 전치(Transpose)에 해당.  
시각적으로: 수평 줄무늬 ↔ 수직 줄무늬 — 모두 Space(XOR)에 속함

### 5.2 단조 함수에서 축 교환

단조 함수 f(A,B,C,D) = AB에 축 교환 적용 → f(C,D,A,B) = CD  
→ CD도 단조 함수 → Space(AND)에 속함

**S₄ 궤도 분석:** 4변수 단조 함수 168개 (Dedekind 수 D(4)) 중  
Space(AND)에 속하는 함수들의 S₄ 궤도 구조 → **FQ-9와 직결**
