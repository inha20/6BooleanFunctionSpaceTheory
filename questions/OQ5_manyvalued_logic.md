## 2. 다치 논리의 기본 개념

### 2.1 k치 논리 (k-Valued Logic)

Boolean = 2치 논리: 값 집합 {0, 1}  
Ternary = 3치 논리: 값 집합 {0, 1, 2} (또는 {-1, 0, 1})  
k치 논리: 값 집합 {0, 1, ..., k-1}

**n변수 k치 논리 함수의 수:**
$$|\mathcal{F}_{n,k}| = k^{k^n}$$

| n | k=2 (Boolean) | k=3 (Ternary) |
|---|---------------|----------------|
| 1 | 4 | 27 |
| 2 | 16 | 19,683 |
| 3 | 256 | 7,625,597,484,987 |
| 4 | 65,536 | 이중 지수적으로 폭발 |

→ 다치 논리에서는 이미 2변수 함수가 수만 개 → 전수 탐구 매우 어려움

### 2.2 Post 격자의 다치 확장

Emil Post(1941)의 원 논문은 2치 논리를 다루었으나, 이후 연구들이 다치 논리로 확장:

- **Rosenberg (1970~80년대):** n치 논리의 클론 격자 분류 (매우 복잡)
- **k=2:** 무한히 많은 클론이나 구조가 분석 가능
- **k≥3:** 클론 격자가 훨씬 더 복잡하고 대부분 미분류

## 3. Boolean에서 Ternary로: 무엇이 확장되는가?

### 3.1 보존되는 구조

| 개념 | Boolean (k=2) | Ternary (k=3) | 확장 가능? |
|-----|---------------|----------------|----------|
| Space(G) 정의 | G-클론 | T-클론 | ✅ (클론 개념 자체는 k-무관) |
| 함수적 완전성 | Post 정리 | Rosenberg 정리 | ✅ (일반화 정리 존재) |
| 변수 치환 대칭 | S₄ 군 | Sₙ 군 | ✅ |
| 폐포 속성 | 합성, 동일화, 상수 대입 | 동일 | ✅ |

### 3.2 달라지는 구조

| 개념 | Boolean | Ternary | 비고 |
|-----|---------|---------|-----|
| Post 클래스 5개 | T₀, T₁, M, L, S | 대응 클래스 존재하나 더 복잡 | 직접 대응 어려움 |
| 카르노 맵 | 2진 직교 격자 (4×4) | 3진 직교 격자 (9×9 for n=2) | 기하학 변화 |
| 아핀 공간 | GF(2) 위 선형 대수 | GF(3) 또는 Z₃ 위 | 대수 구조 변화 |
| 만능 게이트 | NAND, NOR | 달라짐 (Moisil 등) | 연구 필요 |

## 4. Ternary 카르노 맵 (가상적 탐구)

### 4.1 2변수 3치 함수의 카르노 맵 (n=2, k=3)

```
3치 논리 2변수 함수: 입력 (A, B) ∈ {0,1,2}²
진리표 크기: 3² = 9개 입력
함수의 수: 3^9 = 19,683개
```

**3×3 K-map (가상):**
```
      B
      0   1   2
A  0| ?  |  ? | ? |
   1| ?  |  ? | ? |
   2| ?  |  ? | ? |
```

인접성 구조: 각 칸의 인접 칸은 한 변수만 다름  
→ 환면체 위의 격자로 해석 가능하지만, 2진 대신 3진 순환(cyclic) 구조

**문제점:** 3치 카르노 맵은 2치 K-map처럼 직관적이지 않음.  
인접 칸이 "1 비트 차이"라는 Gray 코드 특성을 유지하기 어려움.

### 4.2 진법 확장의 기하학적 의미

2치: K-map = 2차원 torus (T² = (Z₂)² 위의 격자)  
3치: K-map = (Z₃)^n 위의 격자 → "3진 torus"  

→ 기하학은 자연스럽게 확장되지만, 인접성의 직관적 의미(Gray 코드)가 달라짐

## 5. Space(G)의 다치 확장 가능성 분석

### 5.1 3치 NAND의 정의 문제

2치 NAND(A,B) = ¬(A∧B)는 명확히 정의되지만,  
3치 논리에서는 NAND의 정의가 여러 가지로 가능:
- 약 NAND: min(2−A, 2−B) (부정과 최솟값 사용)
- Łukasiewicz NAND: ...

→ 게이트 G의 정의 자체가 k치에서 비유일적이므로 Space(G)도 다양화됨

### 5.2 3치 아핀 공간 (Space(XOR₃))

3치 XOR의 자연스러운 확장: GF(3) 위의 덧셈 (mod 3)

3치 아핀 함수: f(x₁,...,xₙ) = a₀ + a₁x₁ + ... + aₙxₙ (mod 3)  
n=2일 때: a₀, a₁, a₂ ∈ {0,1,2} → 3³ = 27개  
(전체 19,683개 중 27개 = 0.14%)

→ 비율이 더 줄어들지만, 구조는 유사하게 유지됨

### 5.3 보존/소실 요약

| 구조 | 보존 여부 | 이유 |
|-----|---------|-----|
| 클론 개념 자체 | ✅ 보존 | 대수적 정의가 k-무관 |
| 함수적 완전성 정리 | ✅ (일반화) | Rosenberg 정리 적용 |
| 아핀 공간 소수성 | ✅ (더 극단적) | 전체 대비 비율 더 작아짐 |
| K-map 직관성 | ❌ 소실 | Gray 코드 인접성 유지 어려움 |
| Post 5-클래스 분류 | ❌ 소실/변화 | 다치에서 클래스 구조 다름 |
| 만능 게이트 NAND | ❌ 달라짐 | 3치 NAND 정의 비유일적 |

## 6. 연구적 의의와 한계

### 6.1 의의

만약 Space(G) 이론이 3치 논리로 확장된다면:
- 양자 컴퓨팅(qutrit 기반)에의 응용 가능성
- 퍼지 논리(fuzzy logic)와의 연결
- 인지과학적 응용: 인간의 판단이 2치가 아닌 다치적이라면 SRT도 확장 필요

### 6.2 한계

- K-map의 시각적 직관성이 크게 떨어짐
- 3치 함수의 수(k^{k^n})가 폭발적으로 증가 → 전수 탐구 불가
- 다치 Post 격자는 미완성 상태 (수학적으로 훨씬 어려움)
